📖 DB 設定の変更方法

接続するNotionDBを変更するには、GASのスクリプトプロパティを編集します:

① script.google.com を開く →「千石蔵HACCP管理システムv5」を選択
② 左側 ⚙️「プロジェクトの設定」→「スクリプトプロパティ」
③ EXISTING_DBS を編集: 

[
  { "id": "333f5289a51c80f39bf1f02630b63538", "label": "体調管理",       "dateField": "_title_date", "titleProp": "Auto" },
  { "id": "73ff570b9c2a4351b19df2b5451e79bd", "label": "開店前チェック", "dateField": "日付" },
  { "id": "305fa1bec3b94f8483b1adc0688c27c2", "label": "締め作業",       "dateField": "日付" },
  { "id": "2b5f5289a51c8094992ce73a9c59f756", "label": "水質検査",       "dateField": "実施日 Test date" },
  { "id": "90807944c8dc481da413792f96a5b074", "label": "CIP・ATP",       "dateField": "日付" }
]

※ dateField: "_title_date" は、タイトルに手入力されている「YYYY/MM/DD氏名」形式の日付で検索します。 体調管理DBは「作成日時」が入力タイミングのズレ（日付をまたいで入力・後日まとめて入力等）で記録対象日と一致しないことがあるため、 created_time ではなくタイトルの日付を使用します。

※ スクリプトプロパティ変更後、再デプロイは不要です。即時反映されます。

※ 静的書類保管庫（書類本体スプレッドシート）を変更する場合は DOCUMENTS_SPREADSHEET_ID を設定してください。
